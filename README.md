# HSC-Assessment-Task-3-Personal-Project-Stephen-Wirianata
## Overview
SmartInvestments is a simulated investment environment built with Python and CustomTkinter, designed to help users learn the fundamentals of investing in a risk-free setting. Users register an account, deposit virtual paper cash into four investment types, the High Yield Savings Account, Certificate of Deposit, Index Fund, and Stock Market, track performance through live data and graphs, and compete on a leaderboard. 

## Requirements
This program operates on python and CustomTkinter with these modules:
- `customtkinter`
- `pandas`
- `matplotlib`
- `requests`

## Install

```bash
pip install -r requirements.txt
```
